# 结果
各种情况下的断言都是正确的。
```text

['K', '2', 'T', 'Q'] 随机抽四张卡牌

['K', 'Q', 'Q', 'K', '2', '2', 'T'] 因为你是南方人，所以你抽了1张牌插在下面的随机位置

['2', 'T', 'Q', 'K', '2'] 念了一个字'奇'，把牌堆顶部的一张牌放到最下面
['T', 'Q', 'K', '2', '2'] 念了一个字'迹'，把牌堆顶部的一张牌放到最下面

['K', '2', '2', 'T', 'Q'] 念了一个字'时'，把牌堆顶部的一张牌放到最下面
['2', '2', 'T', 'Q', 'K'] 念了一个字'刻'，把牌堆顶部的一张牌放到最下面

['K', '2', 'T'] 说'烦恼丢出去'，把牌堆顶部的一张牌扔掉
['2', 'T', 'K'] 说'好运留下来'，把牌堆顶部的一张牌放到最下面
['T', 'K'] 说'烦恼丢出去'，把牌堆顶部的一张牌扔掉
['K', 'T'] 说'好运留下来'，把牌堆顶部的一张牌放到最下面
['T'] 说'烦恼丢出去'，把牌堆顶部的一张牌扔掉
```
